Installing streamlit-quill
==================

Installing `streamlit-quill` from the main channel can be achieved by:

```
conda install streamlit-quill
```

It is possible to list all of the versions of `streamlit-quill` available on your platform with `conda`:

```
conda search streamlit-quill
```
